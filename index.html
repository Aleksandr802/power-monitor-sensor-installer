ESP32-C3 Super Mini — Веб-інсталятор

Прошивка v1.0.0

📋 Перед початком

  1. 1Підключіть ESP32-C3 Super Mini до цього комп'ютера через USB-кабель для передачі даних (не лише для заряджання).
  2. 2Закрийте Arduino IDE, esptool та інші програми, що використовують послідовний порт.
  3. 3Натисніть Встановити Smart Sensor нижче та виберіть свій пристрій у вікні браузера.
  4. 4Зачекайте 20–40 секунд, поки записується прошивка. Не від'єднуйте пристрій.
  5. 5Готово! Пристрій перезавантажиться автоматично.
Працює лише у Chrome або Edge на комп'ютері. Мобільні браузери та Firefox не підтримуються.

⚡ Встановити прошивку

Підключається через WebSerial і прошиває bootloader + partitions + firmware за один крок.

Чип: ESP32-C3 Super Mini  |  Розмір: ~1.5 MB

🔄 Оновлення прошивки (OTA)

Оберіть зручний спосіб OTA-оновлення. На пристрої вже має бути встановлена ця прошивка.

Рекомендовано Спосіб 1 — Оновлення через USB (ESP Web Tools)

Так само, як і перше встановлення: підключення через USB-кабель і перепрошивка лише розділу firmware.

Wi-Fi Спосіб 2 — OTA через локальну мережу

Відкрийте вбудовану сторінку OTA-оновлення пристрою у вашій локальній мережі. Пристрій має бути підключений до Wi-Fi.

IP-адресу можна знайти в адмінпанелі роутера або в послідовному виводі пристрою.

📦 Що саме оновлюється?

  1. 1Замінюється лише розділ застосунку (firmware.bin) — bootloader і налаштування зберігаються.
  2. 2Wi-Fi облікові дані та конфігурація пристрою не видаляються.
  3. 3Після оновлення пристрій автоматично перезавантажується в нову прошивку.
🔌 Послідовний порт не знайдено / пристрій не відображається
  • Спробуйте інший USB-кабель — багато кабелів призначені лише для заряджання і не мають ліній даних.
  • Встановіть драйвер CH340 / CP210x для вашої ОС (ESP32-C3 Super Mini використовує CH340G).
  • У Linux додайте себе до групи dialout: sudo usermod -aG dialout $USER, потім вийдіть і увійдіть знову.
  • Закрийте всі програми моніторингу порту (Arduino IDE, minicom, screen).
⚠️ Прошивка стартує, але завершується помилкою або таймаутом
  • Утримуйте кнопку BOOT на ESP32-C3 Super Mini під час натискання «Встановити», відпустіть після початку прогресу.
  • Спробуйте коротший USB-кабель або інший USB-порт на комп'ютері.
  • Зменште baud rate, якщо бачите часті помилки (ESP Web Tools зазвичай робить це автоматично).
  • Переконайтеся, що пристрій не в deep sleep — спочатку натисніть кнопку reset.
🌐 Браузер показує "WebSerial не підтримується"
  • Використовуйте Google Chrome 89+ або Microsoft Edge 89+ на Windows, macOS чи Linux (десктоп).
  • Firefox, Safari та мобільні браузери не підтримують WebSerial.
  • Для старих версій Chrome перевірте, що ввімкнено chrome://flags/#enable-experimental-web-platform-features.
💡 Пристрій підключений, але не відповідає після встановлення
  • Натисніть кнопку RST (reset) на платі для примусового перезавантаження.
  • Відкрийте монітор порту на 115200 baud, щоб побачити повідомлення завантаження.
  • Якщо пристрій у bootloop, перепрошийте через вкладку Перше встановлення (повна прошивка).
